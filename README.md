# Math Career Guidance 2026 Presentation

## How to Run

1. Open `index.html` in your browser, or run a local server:
   ```bash
   python3 -m http.server
   ```
   Then open http://localhost:8000 in your browser.

2. Use arrow keys or spacebar to navigate slides.

## Structure

- `index.html` — Main presentation file
- `slides/` — All slide HTML files (Title.html, AI_architect.html, etc.)
- `README.md` — This documentation
